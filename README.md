# 关于
整合SSM框架（SpringMVC + Spring + MyBatis），适合刚接触spring的童鞋，需要有servlet和jsp基础。本文是一个简单的SSM整合案例，基于RESTful风格，主要是简单记录一下学习笔记。学习完SSM整合后可以应付开发工作，但是建议继续深究，学习spring boot，spring cloud等技术，让开发效率更上一层楼。
